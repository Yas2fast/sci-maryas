SCI MARYAS — Web app simple de gestion

Fonctions incluses :
- création et modification de clients
- création de factures
- création de quittances de loyer
- suivi payé / impayé
- aperçu imprimable pour PDF
- export / import des données
- stockage local dans le navigateur

Utilisation :
1. Ouvrir index.html dans un navigateur.
2. Remplir Paramètres.
3. Ajouter des clients.
4. Créer les documents.
5. Cliquer sur “Voir”, puis “Imprimer / PDF” pour sauvegarder en PDF.

Important :
- Cette version est une base locale simple, sans serveur.
- Les données sont stockées dans le navigateur via localStorage.
- Pour une vraie version professionnelle en ligne, il faudra ensuite :
  - un hébergement
  - une vraie base de données
  - des comptes utilisateurs
  - des sauvegardes serveur
  - éventuellement un envoi email manuel/intégré
